# Thunder Fighter (雷霆战机)

C++ 课程大作业 — 基于 Windows GDI+ 的纵版射击游戏。

## 运行方式

使用 MinGW-w64 编译：

```bash
mingw32-make
./ThunderFighter.exe
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ↑↓←→ / WASD | 移动 |
| J | 射击 |
| K | 炸弹 |
| Enter | 确认 |
| Space | 暂停 |
| Esc | 返回 |
